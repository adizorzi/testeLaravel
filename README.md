
# Instalação

```
Git clone:
https://github.com/adizorzi/testeLaravel.git

Abra a pasta do projeto
cd testeLaravel

Atualize o Composer
composer update

Configure o banco de dados no arquivo .env

Rode as Migrations
php artisan migrate

Rode os Seeders
php artisan db:seed

Rode a aplicação
php artisan serve

```
